\doxylink{class_m_p3}{MP3} player folosind clase si diferite librarii pentru functionalitatea sa. Principala librarie folosita este SDL2 pentru audio playback. In acest moment, aplicatia este capabila sa incarce diferite melodii salvate local, sa le redea, play/pause/skip. Utilizatorii pot creea playlisturi si pot folosi functio shuffle.

Acest program este simplu dar eficient.\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Milestone \#0}{Milestone \#0}}\label{index_autotoc_md2}
